# Yasree Works — E-commerce Static Site

This repository contains product pages and images for the Yasree Works demo ecommerce site.

## Run locally
python -m http.server 8000
Open http://127.0.0.1:8000/maincode.html

## Notes
- Images should be kept in an `images/` folder with simple filenames (kebab-case).
- Use .gitignore to avoid committing node_modules, editor files, etc.
